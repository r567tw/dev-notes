## Wordpress 也想要 hot-reload
由於最近在開發前端相關的程式，但也希望開發`WP`時也能像開發`gulp`、`golang`一樣能夠 hot-reload。

問了`google`，它就推薦了`browser-sync` 這個套件，實在是好好用啊～

首先，讓我們先安裝他
```bash 
npm install browser-sync --save-dev
```

之後你需要在目錄下寫一個`config.js`，以下是我的配置
```javascript
module.exports = {
    proxy: 'http://127.0.0.1', // 將此處替換為你的本地 WordPress 網站 URL
    files: ['./**/*'], // 將此處替換為你的插件檔案目錄路徑
    reloadDelay: 1000, // 可選：重新載入延遲（以毫秒為單位）
};
```

在 `npm install` 之後你編輯一下`package.json`
```json
{
  "scripts": {
    "serve": "npx browser-sync start -c config.js"
  },
  "devDependencies": {
    "browser-sync": "^2.29.3"
  }
}
```

之後執行`npm run serve` 就可以啦！ 💪
