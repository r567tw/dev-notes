# 原來可以直接創造一個class...

```php
trait TraitA {
     public    function   HelloWorld(){
           echo'Hello World';
      }
}
trait TraitB {
   public   function   HELLOTWO(){
       echo'hELLO tWO ';
    }
}
$helper = new class {
      useTraitA,TraitB;
};
$helper->HelloWorld();
```
