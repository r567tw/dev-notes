## Python 也可以在 Html 上執行
看到這個有趣的專案：https://pyscript.net/
他透過將 Python 解釋器轉換成 WebAssembly 形式，從而讓 Python 代碼能夠在瀏覽器端運行

讓我不經想著，或許有一天 PHP 也可以在 Client 執行這樣ＸＤ

## it-tool
在網路上看到一個有趣的網站：https://it-tools.tech/

裡面搜集了各式各樣工具，根本就是我想要的那種工具箱啊（推眼鏡

還可以簡單用`docker`給他架起來，於是我也很快就把他加入到我的`docker-compose.yml`囉
```bash
docker run -d --name it-tools --restart unless-stopped -p 9999:80 corentinth/it-tools:latest
```
