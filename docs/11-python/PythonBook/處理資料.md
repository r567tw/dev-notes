## 第七章 喔喔：處理資料
- 文字字串

    - Unicode
    - UTF-8
- encode() , decode()
- 用 `%` 格式化字串
- 用 `{}`與format 
```
'{} {} {}'.format(n,f,s)
'{2} {0} {1}'.format(s,n,f)
'{n} {f} {s}'.format(n=1,f=7,s='string')
'{n:d} {f:f} {s:s}'.format(n=1,f=7,s='string')
```
