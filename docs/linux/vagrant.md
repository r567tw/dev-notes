## Install

- https://developer.hashicorp.com/vagrant/install
- Mac For example

```bash
brew tap hashicorp/tap
brew install hashicorp/tap/hashicorp-vagrant
```

## Confirm

```bash
vagrant version
```

## init

```bash
# in dir (for example workspace)
vagrant init ubuntu/focal64
vagrant up
```
